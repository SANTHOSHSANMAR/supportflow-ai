# SupportFlow AI

AI-powered customer support automation system built as a proof-of-work project.

## Features

- Ticket classification
- Priority detection
- Suggested customer replies
- Analytics dashboard

## Tech Stack

- Python
- Streamlit
- Pandas

## Run Locally

pip install -r requirements.txt  
python -m streamlit run app.py

## Project Goal

Reduce response time, improve customer satisfaction, and automate repetitive support tickets.
